# Canyon Assets

This repository contains static assets for the Canyon project, including documentation images, logos, screenshots, and other media files.

## Structure

- `docs/` - Documentation assets including images, logos, and screenshots
  - `static/docs/` - Organized documentation assets by category
    - `ecosystem/` - Ecosystem-related images
    - `getting-started/` - Getting started guide assets
    - `self-host/` - Self-hosting documentation assets

## Usage

These assets are used by the Canyon project documentation and other related resources. They are organized by topic and purpose for easy reference and maintenance.
